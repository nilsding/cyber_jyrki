# cyber_jyrki

A telegram bot.

## Installation

build it:

```
shards build
```

first-time configuration:

```sh
# create environment file containing the configuration
echo "TELEGRAM_API_KEY=1234567890:abcdefghijklmnop..." > .env
```

## Usage

once built:

```sh
# run the bot
./bin/cyber_jyrki
```
